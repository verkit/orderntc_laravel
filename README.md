## Instalasi

```
composer install
```

- Salin contoh environment bawaan
```
cp .env.example .env
```
- buka file .env dan setting databasenya

- setelah mengkonfigurasi environment generate key-nya
```
php artisan key:gen
```

- Web siap dijalankan
```
php artisan serve
```
